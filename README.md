# Schedule Clinic

## Tecnologias

**Backend:** Java + Spring Boot 
**Frontend:** React
**Banco de Dados:** PostgreSQL
**Build Backend:** Maven 
**Build Frontend:**  npm
**Versionamento:**  Git + GitHub 
**CI/CD:** GitHub Actions 
**Containers:**  Docker
**Produção:** AWS 

## Estrutura do Projeto

```text
schedule-clinic/
├── backend/          
│   ├── pom.xml
│   ├── Dockerfile
│   └── src/
├── frontend/      
│   ├── package.json
│   ├── Dockerfile
│   └── src/
├── docker-compose.yml
├── docs/
└── apresentacao_completa.html
```

### Iniciar

```bash
cd schedule-clinic
docker-compose up -d --build
```

### Acessando

**[http://localhost:3000](http://localhost:3000)**

### Desligar

```bash
docker-compose down
```
